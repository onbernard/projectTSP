Parses a tsp file, returns the result in an instance_t structure. IT IS VERY STRICT ON THE FILE FORMAT.

Parameters
fp: a file pointer to the tsp file
instance: buffer to wich the result are written
nz: boolean value : 0 if the (0,0) node has to be is added, 0 if not
Returns
a negavive value if an error has occured, 0 otherwise

Parses a tsp file. It allocates instance->tabTour and the two dimensionnal array instance->tabCoord CALLS ABORT ON ALLOCATION ERROR If nz is 0 then a point with coordinates (0,0) is automatically placed on the first index and dimension is incremented by one Right now these are accepted: ::: firsthalf ::::: second half ::: fzefzeafeza Only EUC_2D edges type and TSP file type are accepted Integers must be strictly greater than 0 A city number must be equal to its position
