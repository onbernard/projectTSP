Generates the next permutation of integer sequence in lexicographic order. Tabtour is overwritten if there is a next one after it.

Parameters
tabTour: A pointer to the array containing the permutation of cities
size: The size of the tabTour array
Returns
A boolean value. 1 if there is no next permutation, 0 otherwise and tabTour is overwritten

Takes an array filled with integers from 0 to size-1 and generate the next permutation in the lexicographic order. If the permutation passed is the last one, it does not change it and returns 1. Otherwise, it overwrites tabTour and returns 0;
