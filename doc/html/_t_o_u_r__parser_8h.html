Parses a tour file, returns the result in an instance_t structure. IT IS VERY STRICT ON THE FILE FORMAT.

Parameters
fp: a file pointer to the tour file
instance: buffer to wich the result are written
nz: boolean value : 0 if the (0,0) node has to be is added, 0 if not
Returns
a negavive value if an error has occured, 0 otherwise

Parses a tour file and stores the results in the instance_t structure pointed by argument instance If nz is 0 then the point (0,0) is automaticly added at the first index It allocates instance->tabTour CALLS ABORT ON ALLOCATION ERRORS
